### 3.1

Q：读写磁盘文件时，本章描述的函数确实是不带缓冲的吗？

A： 准确说是不带用户缓冲。read/write函数每次都执行系统调用，放入系统缓冲区中。等系统缓冲区满了，刷新入文件。fread/fwrite则先放入用户缓冲，满了再进行系统调用

### 3.5

区别：
```
./a.out > outfile 2>&1  (1)
./a.out 2>&1 >outfile   (2)
```
bash从左到右执行，有先后顺序
(1) 会把标准输出和标准错误输出到outfile。相当于dup2(fd_outfile, 1); dup2(1,2);
(2) 标准错误会输出到控制台，标准输出输出到outfile。相当于dup2(1, 2); dup2(fd_outfile, 1);
